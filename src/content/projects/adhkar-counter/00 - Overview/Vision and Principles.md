# Vision and Principles

## Vision

A small, calm macOS utility that helps a Muslim user make remembrance of Allah part of the quiet background of their work. No gamification, no noise, no dashboards — just a number that goes up, beautifully.

## Principles

1. **Privacy-first**. No network, no telemetry, no accounts, no cloud sync. Local `UserDefaults` only in v1. This is a product promise, not a checkbox.
2. **Do no harm**. Destructive actions (reset) always confirm. No patterns that push engagement. No streaks that shame you.
3. **Effortless increment**. The primary action is the easiest possible: a large click target, plus an optional global keyboard combo that works from any app. Decrement is not first-class.
4. **Quiet by default**. Accent colour is used sparingly. Motion is barely perceptible. Sound and haptics are absent.
5. **Apple-native feel**. Typography rhythm, materials, radii, spacing and window behaviour match what the platform expects of a utility widget.
6. **Honesty over illusion**. We tell the user exactly what works: a modifier combo fires globally with no permission prompt; a plain key would require system-wide key observation and is deliberately off by default. No hand-waving in the UI copy.
7. **Small surface area**. Features are added only when they serve remembrance, not because they are technically possible.
