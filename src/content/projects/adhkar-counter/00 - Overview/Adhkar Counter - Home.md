# Adhkar Counter — Home

A tiny, always-available macOS counter for dhikr. The primary interaction is effortless: a large click target and an optional keyboard shortcut. The app is privacy-first — no network, no analytics, no accounts.

## What it is

- A borderless floating panel that stays above other windows without stealing focus.
- A quiet numeral that increments with a click, or with a modifier-bearing keyboard combo that works from any app.
- Settings in a native macOS Preferences-style window (`⌘,`).

## What it is not

- A gamified streak tracker.
- A dashboard with metrics.
- A cloud-synced, cross-device habit app.
