# Credits

Every external source — software, mathematics, paper, art — that the
project uses or that informed the design. One file per source.

The format for each entry: **what the source is**, **how this project
uses it**, and **the citation/link**.

## Software dependencies

- [[manim_community]] — the rendering engine. Manim Community Edition
  0.20.1.
- [[wallspace_app]] — the macOS wallpaper player the renders target.

## Mathematical sources

- [[vogel_phyllotaxis]] — Vogel's 1979 closed-form model for sunflower
  phyllotaxis. Used directly by [[wallpapers/03_phyllotaxis]].
- [[lindenmayer_lsystems]] — L-system formalism (Aristid Lindenmayer,
  1968). Conceptually inspired [[wallpapers/01_recursive_tree]], though
  the implementation uses straight recursion rather than a formal
  L-system grammar.

## What this list deliberately omits

- Internet-found wallpapers that happen to share an aesthetic — no
  specific image was copied or remixed.
- General-knowledge animation conventions (sine sway, k-nearest graph
  edges, raised-cosine easing) — these are common-domain techniques
  rather than attributable to a single author.

If you spot a credit-worthy source that should be here, the
contribution is welcome.
