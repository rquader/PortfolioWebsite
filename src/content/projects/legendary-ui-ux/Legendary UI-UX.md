# legendary

> a small site, a long thought.
> field notes from a quieter web.

This vault is the design journal for a single self-contained HTML artefact: a manifesto-as-experience structured as **four rooms** (tabs). One file. No network. No fonts to fetch. ~75 KB.

**Provenance:** the site began as an experiment in what AI could build in one file with no dependencies. Most of the code and copy are machine-assisted, shaped with feedback from a human. The manifesto now includes an **on making** section (after the six movements, before the coda) that says this plainly. This vault is part of the same experiment — largely AI-drafted notes, human feedback throughout.

The live artefact lives at `~/Developer/Legendary-UI-UX/index.html`. Drop it in any modern browser. (Or `open` it from a terminal — `open ~/Developer/Legendary-UI-UX/index.html`.)
